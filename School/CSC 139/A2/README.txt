heap.c - A simple heap implementation for SRTF algorithm
heap.h - header file for heap
queue.c - A simple circular queue array implementation for RR and FCFS algorithm
queue.h - header file for queue
proj2.c - The main file containing scheduling algorithm implementations and file input

HOW TO RUN:
compile:
gcc proj2.c heap.c queue.c -o proj2

run:
proj2 input_file [FCFS|RR|SRTF] [time_quantum]
